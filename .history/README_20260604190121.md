# AI Numerasi Siswa Tuli

Platform asesmen numerasi untuk siswa tunarungu berbasis Laravel 12.
Aplikasi ini dirancang untuk mengukur kemampuan numerasi siswa melalui serangkaian soal pilihan ganda dan menghasilkan diagnosis berbasis pola indikator jawaban.

## Fitur Utama

- Dashboard manajemen hasil asesmen dan statistik global.
- Alur pelaksanaan asesmen: pilih siswa/sekolah, kerjakan soal, dan tampilkan hasil diagnosis.
- Penyimpanan riwayat siswa, sesi asesmen, jawaban, dan hasil diagnosis.
- Diagnosis berbasis indikator dari setiap pilihan jawaban.
- Halaman daftar siswa dan detail riwayat asesmen.

## Teknologi

- Laravel 12
- PHP 8.2+
- MySQL / SQLite / PostgreSQL (konfigurasi database dari `.env`)
- Vite + Tailwind CSS untuk asset frontend
- Axios, Laravel Vite Plugin

## Struktur Data Penting

- `Student` — data siswa tunarungu.
- `School` — sekolah asal siswa.
- `Question` / `Option` — bank soal numerasi dan pilihan jawaban.
- `AssessmentSession` — sesi asesmen berjalan.
- `Answer` — jawaban siswa pada setiap soal.
- `DiagnosisResult` — hasil diagnosis akhir.

## Routes Utama

- `/` — halaman dashboard.
- `/assessment/start` — mulai asesmen baru.
- `/assessment/{token}/soal/{no}` — halaman soal per sesi.
- `/assessment/result/{result}` — halaman hasil diagnosis.
- `/students` — daftar siswa.
- `/students/{id}/detail` — detail riwayat siswa.
- `/statistics` — ringkasan statistik keseluruhan.

## Persiapan Lokal

1. Salin file lingkungan:

    ```bash
    cp .env.example .env
    ```

2. Pasang dependensi PHP:

    ```bash
    composer install
    ```

3. Jalankan migrasi database:

    ```bash
    php artisan migrate
    ```

4. Isi data awal (sekolah dan soal):

    ```bash
    php artisan db:seed
    ```

5. Pasang dependensi JavaScript:

    ```bash
    npm install
    ```

6. Bangun asset frontend:

    ```bash
    npm run build
    ```

7. Jalankan aplikasi lokal:

    ```bash
    php artisan serve
    ```

> Jika menggunakan database SQLite, buat file `database/database.sqlite` sebelum migrasi.

## Pengembangan

- `npm run dev` — jalankan Vite dalam mode development.
- `php artisan test` — jalankan test Laravel.
- `php artisan migrate:fresh --seed` — reset database dan isi ulang seed.

## Catatan Khusus

- Soal diagnostik numerasi ditujukan untuk siswa tunarungu.
- Setiap pilihan jawaban membawa indikator kemampuan yang dipakai oleh `DiagnosisService` untuk menghitung level, akurasi, dan rekomendasi.
